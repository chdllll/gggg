# AI角色扮演软件

一款基于PySide6和DeepSeek API的AI角色扮演软件，支持自定义世界、角色、背景图，并提供智能对话功能。

## 功能特点

- **世界管理**：创建和编辑自定义世界，设置背景、叙事风格和特殊背景
- **角色管理**：为每个世界创建多个角色，设置背景、描述、对话示例、头像和背景图
- **智能对话**：通过DeepSeek API生成自然的角色对话，支持多角色互动
- **背景图选择**：AI根据对话内容自动选择合适的背景图
- **记忆系统**：自动提取和存储重要记忆，保持对话连贯性

## 系统要求

- Python 3.8+
- Windows 10/11
- 4GB+ RAM

## 安装步骤

1. 克隆或下载本项目到本地

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行程序：
```bash
python main.py
```

## 使用说明

### 首次使用

1. **设置API密钥**
   - 点击主界面的"⚙️ API设置"按钮
   - 输入DeepSeek API Key（必需）
   - 选择DeepSeek模型（默认：deepseek-chat）
   - 点击"保存"

2. **创建世界**
   - 点击"➕ 创建世界"按钮
   - 输入世界名称
   - 在"世界设置"标签页设置：
     - 背景设定：描述这个世界的基本背景
     - 叙事风格：描述故事的叙事风格
     - 特殊背景设定：用于在讨论敏感话题时获得更大的创作自由
   - 点击"💾 保存"

3. **创建角色**
   - 在世界编辑对话框中切换到"角色管理"标签页
   - 点击"➕ 添加角色"按钮
   - 输入角色名称
   - 在"基本信息"标签页设置：
     - 背景：角色的背景故事
     - 描述：角色的性格、外貌、习惯等
     - 对话示例：帮助AI理解角色的说话风格
   - 在"头像"标签页上传和裁剪头像
   - 在"背景图"标签页添加角色背景图
   - 点击"💾 保存"

4. **开始角色扮演**
   - 返回主界面，选择一个世界
   - 点击"🚀 进入世界"按钮
   - 在聊天界面输入消息，开始角色扮演

### 对话规则

- 用户和角色的对话都遵循相同规则：
  - 括号外是对话内容
  - 括号内是动作或发生的小事情
  - 例如：`(微笑) 你好，很高兴见到你`

- 多个角色可以在同一轮对话中先后发言
- 角色之间可以进行对话互动

## API配置

### DeepSeek API

- 获取API Key：访问 https://platform.deepseek.com/
- 支持的模型：
  - deepseek-chat：通用对话模型
  - deepseek-reasoner：推理模型，适合复杂推理任务

## 注意事项

1. **API密钥安全**：请妥善保管API密钥，不要泄露给他人
2. **数据备份**：定期备份`data/roleplay.db`数据库文件
3. **网络连接**：使用DeepSeek API需要稳定的网络连接

## 常见问题

### Q: 如何获取DeepSeek API Key？
A: 访问 https://platform.deepseek.com/ 注册账号并获取API Key。

### Q: 对话不自然？
A: 可以尝试：
- 完善角色的背景和描述
- 提供更多对话示例
- 调整叙事风格
- 设置特殊背景以获得更大的创作自由

### Q: 如何删除世界或角色？
A: 在主界面选择世界后，点击"🗑️ 删除"按钮。角色删除同理。
